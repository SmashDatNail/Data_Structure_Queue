# Data_Structure_Queue
An Arduino project that demonstrates how a real-world restaurant ordering system works using the **Queue data structure**.
There are two ways for you can do to implement this system:
1. Arduino with LCD
2. ESP32 with webserver.h (with wifi capabilities)
## Project Overview
This project simulates a restaurant order management system. Customers can place food orders by pressing buttons, and the kitchen processes the orders based on the **First In, First Out (FIFO)** concept.
The first order entered will be the first order prepared.
